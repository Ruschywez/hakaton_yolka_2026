## 📖 Описание

Клиентское приложение **Ёлка** — это десктопное приложение с современным тёмным интерфейсом, которое предоставляет удобный доступ к AI-ассистенту для карьерного роста. Приложение разработано на **Tkinter** и взаимодействует с backend-сервером через REST API.

### Ключевые особенности:

- 🔐 **Авторизация и регистрация** — быстрый вход или создание аккаунта
- 💬 **Чат с AI-ассистентом** — реальное время, индикаторы набора текста
- 👤 **Профиль пользователя** — просмотр и редактирование данных
- 🎨 **Тёмная тема** — современный дизайн с акцентными цветами
- 📱 **Адаптивность** — минимальные размеры окна для удобства

## ⚙️ Настройка

### Подключение к серверу

Откройте файл `config.py` и укажите адрес вашего API-сервера:

```python
# API
API_BASE_URL = "http://26.164.106.34:8000/"  # Замените на ваш адрес
MOCK_MODE = False  # True для работы без сервера (демо-режим)

# Приложение
APP_TITLE = "Ёлка — Карьерный AI Ассистент"
ASSISTANT_NAME = "ВолодяGPT"
ASSISTANT_DESCRIPTION = "Твой персональный карьерный ассистент"

# Окно
WINDOW_WIDTH = 1100
WINDOW_HEIGHT = 700
MIN_WINDOW_WIDTH = 900
MIN_WINDOW_HEIGHT = 600
```

### Параметры конфигурации

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `API_BASE_URL` | Базовый URL сервера API | `http://26.164.106.34:8000/` |
| `MOCK_MODE` | Режим без сервера (демо-данные) | `False` |
| `APP_TITLE` | Заголовок окна | `Ёлка — Карьерный AI Ассистент` |
| `ASSISTANT_NAME` | Имя AI-ассистента | `ВолодяGPT` |
| `ASSISTANT_DESCRIPTION` | Описание ассистента | `Твой персональный карьерный ассистент` |
| `WINDOW_WIDTH` | Ширина окна | `1100` |
| `WINDOW_HEIGHT` | Высота окна | `700` |
| `MIN_WINDOW_WIDTH` | Минимальная ширина | `900` |
| `MIN_WINDOW_HEIGHT` | Минимальная высота | `600` |

### Демо-режим (без сервера)

Для тестирования интерфейса без подключения к серверу:

```python
MOCK_MODE = True
```

В этом режиме используются моковые данные для авторизации и сообщений.

## 🖥️ Интерфейс приложения

- Поля: Логин, Пароль
- Кнопка «Войти»
- Ссылка «Зарегистрироваться»
- Отображение ошибок авторизации

  - **Правая колонка:** Отчество, Дата рождения
- Кнопка «Зарегистрироваться»
- Ссылка «Войти» (возврат на экран входа)

  - Логотип «Ёлка»
  - Приветствие пользователя
  - Навигация: Чат, Профиль, Изменить данные
  - Кнопка «Выйти»
- **Заголовок чата:** Статус (онлайн), Имя ассистента
- **Область сообщений:** Прокручиваемый список пузырей сообщений
- **Область ввода:** Поле ввода + кнопка «Отправить»
- **Индикатор набора:** Анимация при ожидании ответа от AI

#### 4. Экран профиля (Profile Screen)
- **Аватар:** Круг с инициалами пользователя
- **ФИО:** Полное имя пользователя
- **Информация:**
  - Дата рождения
  - Образование
  - Специальность
  - Интересы
- **Кнопка:** «Редактировать профиль»

#### 5. Экран редактирования профиля (Edit Profile Screen)
- **Двухколоночная форма:**
  - **Левая колонка:** Фамилия, Имя, Отчество, Дата рождения
  - **Правая колонка:**
    - Уровень образования (выпадающий список)
    - Специальность
    - Увлечения
    - Новый пароль
- **Кнопки:** «Сохранить», «Отмена»
- **Статус сохранения:** Сообщение об успехе/ошибке

## 🔌 Структура API

Клиент взаимодействует с сервером через следующие эндпоинты:

### Авторизация

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| `GET` | `/authorization/?login={}&password={}` | Вход пользователя |
| `POST` | `/authorization/` | Регистрация |
| `PATCH` | `/authorization/` | Обновление профиля |
| `DELETE` | `/authorization/{user_id}` | Удаление аккаунта |

### Сообщения

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| `GET` | `/messages/?user_id={}` | Получить историю сообщений |
| `POST` | `/messages/` | Отправить сообщение |
| `PATCH` | `/messages/` | Редактировать сообщение |
| `DELETE` | `/messages/{message_id}` | Удалить сообщение |

### Формат запросов

```python
# Авторизация
client.get('authorization/', params={'login': 'user', 'password': 'pass'})

# Регистрация
client.post('authorization/', data={
    'login': 'user',
    'password': 'pass',
    'last_name': 'Иванов',
    'first_name': 'Иван',
    'sur_name': 'Иванович',
    'date_birth': '2005-06-15'
})

# Обновление профиля
client.patch('authorization/', data={
    'user_id': 1,
    'interests': 'Python, AI'
})

# Отправка сообщения
client.post('messages/', data={
    'user_id': 1,
    'text': 'Привет!'
})
```

## 📦 Сборка исполняемого файла

Для создания standalone `.exe` файла используйте **PyInstaller**:


## 🐛 Решение проблем

### Ошибка подключения к серверу

**Симптом:** `GET authorization/: HTTPConnectionPool...`

**Решение:**
1. Проверьте `API_BASE_URL` в `config.py`
2. Убедитесь, что сервер запущен и доступен
3. Проверьте сетевое подключение

### Tkinter не установлен

**Симптом:** `ModuleNotFoundError: No module named 'tkinter'`

**Решение:**

```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# CentOS/RHEL
sudo yum install python3-tkinter

# Windows/macOS
# Tkinter встроен в установщик Python
```

### Пустой экран или ошибки UI

**Решение:**
1. Проверьте версию Python (должна быть 3.10+)
2. Запустите с флагом отладки:
   ```bash
   python -u main.py
   ```
